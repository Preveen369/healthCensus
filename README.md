# Health Census
Health Census is a static web application designed to provide users with interactive health assessments and information on common conditions such as blood pressure, diabetes, and thyroid issues. It features dynamic analysis tools, educational content with images, and a contact form for inquiries, all built with vanilla web technologies for easy access and offline use.
